## 📝 使用说明

### 启动项目
```bash
# 1. 安装依赖
npm install --legacy-peer-deps

# 2. 启动后端 (在后端目录)
python start_web_service.py

# 3. 启动前端
npm run dev

# 4. 打开浏览器
http://localhost:3000
```

### 构建生产版本
```bash
npm run build
npm run start
```
